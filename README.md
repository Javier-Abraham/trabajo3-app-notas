Trabajo Práctico 3 - Aplicación de Notas

Descripción
Aplicación desarrollada en Android con Kotlin y Jetpack Compose.

Permite:
- Ingresar nombre del alumno
- Ingresar dos notas
- Calcular promedio
- Generar nota aleatoria de participación
- Validar datos ingresados

Tecnologías usadas
- Kotlin
- Jetpack Compose
- Android Studio

Funcionalidades
- Validación de campos vacíos
- Validación de datos numéricos
- Validación de rango (0 a 10)
- Cálculo de promedio
- Estado: aprobado / desaprobado
